# 项目操作记录

## 2024-03-21 后端基础框架搭建

### 完成工作
1. 创建了基础的 Flask 应用程序 (app.py)
   - 实现了基础的健康检查接口
   - 配置了CORS支持
   - 添加了首页路由

2. 配置了项目依赖 (requirements.txt)
   - Flask 3.0.0
   - flask-cors 4.0.0
   - gunicorn 21.2.0
   - PyMySQL 1.1.0
   - python-dotenv 1.0.0

3. 创建了 gunicorn 配置文件 (gunicorn_conf.py)
   - 配置了工作进程和线程
   - 设置了日志路径
   - 配置了端口绑定

### 部署说明
1. 安装依赖：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

2. 启动服务：
```bash
gunicorn -c gunicorn_conf.py app:app
```
